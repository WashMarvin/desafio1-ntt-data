Estrutura Básica do HTML

(Voltar)

HTML significa Hypertext markup language. Em tradução para o português, Linguagem de Marcação de Hipertexto. A primeira informação necessária é que HTML não é uma linguagem de programação e tem como finalidade é dar significado e organizar as informações de um website. A marcação é utilizada para marcar textos, imagens e diversos contéudos para exibi-los em um navegador. O hipertexto está ligado aos links de conexão entre páginas web dentro de um único site ou entre sites.

Doctype

O Doctype é uma instrução a ser compreendida pelo navegador informando-o que o código encontrado ali é HTML. O Doctype é obrigatório e é o primeiro a ser carregado no seu documento.

Head

Na tag Head as informações não são visualizadas pelo usuário. São dados implícitos de uso e controle do documento. Onde residem metadados, vínculo com outros arquivos, aplicação de lógica e programação de scripts. Em resumo, todo o conteúdo de cabeçalho se encontram entre a abertura e fechamento desta tag.

body

Aqui residem as informações visíveis para o usuário. Todo conteúdo que se deseja apresentar, como textos, mídias, imagens, etc. O corpo de documento é delimitado pelo par de tag body. Dentro do body, a estrutura da página deverá possuir os elementos semânticos em que serão declarados cabeçalhos, conteúdo, rodapé, entre outros.

Tags

(Voltar)

É um conjunto de caracteres que tem como função formar um elemeto. Existem dois tipos de tags, as que necessitam de fechamento e as que não necessitam.

Elementos

Elementos são formados a partir de tags. Entre essas tags há o conteúdo do elemento. As tags servem para marcar onde começam e terminam os elementos, o elemento é uma parte semântica que tem começo e fim determinados.

Atributos

(Voltar)

São informações que passamos nas tags para que ela se comporte da maneira desejada. Existem atributos globais (que funcionam em todas as tags) e específicos (que são direcionados para cada tag, através de especificação).

Os atributos possuem nome e valor. Alguns você vai usar praticamente o tempo todo e outros que serão mais raros.

Exemplos de alguns atributos

accesskey

Determina uma ou mais teclas de atalho para o elemento.

class

Determina uma ou mais classes para o elemento. Para definir mais de uma classe, coloque-as separadas por espaço.

draggable

Define se um elemento é arrastável ou não.

hidden

Oculta o elemento onde for declarado.

id

É o identificador único do elemento. Somente deve ser declarado um id por elemento e este não deve ser repetido na mesma página.

Listas

(Voltar)

O HTML tem elementos especiais para listas. Consistem de dois elementos que podem ser ordenadas e não ordenadas. Cada item dentro da lista é posto dentro de um elemento li.

Listas ordenadas

São listas onde a ordem dos itens importa, como uma receita de bolo, por exemplo. Elas estão envolvidas em um elemento "ol".

Listas não ordenadas

São listas onde a ordem dos itens não importa, como uma lista de compras, por exemplo. Estão envolvidas pelo elemento ul.

Links

(Voltar)

Os links determina a relação entre documentos internos e também externos. Este elemento é utilizado para vincular também as folhas de estilos.

Os links são definidos pela tag a e dentro desta tag incluimos o atributo href (Hypertext Reference), que é o endereço para o qual se destina o link.
